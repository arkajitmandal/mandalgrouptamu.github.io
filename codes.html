Codes/Software


Cavity on-the-fly (CavOTF) 


Allows performing cavity molecular dynamics simulations on-the-fly. Uses Density Functional Tight-Binding (DFTB) approach and uses DFTB+ for propagating the molecular part.

Semiclassical Quantum Dynamics (SQD)


Code for performing Semi-Classical Quantum Dynamics. The methods that are implimented in this code are : PLDM, spin-PLDM, spin-LSC, MFE, various SQC, MASH, and N-RPMD.
